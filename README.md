📱 Описание приложения
Dostaffkin — это мобильное Android-приложение для управления заказами доставки. Оно позволяет пользователям создавать заказы на отправку посылок, отслеживать их статус и управлять списком своих отправлений. Приложение полностью автономно и сохраняет данные локально на устройстве пользователя.
Основные функции:
✅ Главный экран: Отображение списка всех созданных заказов.
✅ Создание заказа: Форма ввода маршрута (Откуда/Куда), выбора размера и скорости доставки.
✅ Просмотр деталей: Карточка заказа с текущим статусом (например, "В пути").
✅ Редактирование: Возможность изменить параметры существующего заказа.
✅ Удаление: Удаление заказа с подтверждением.
✅ Валидация: Проверка обязательных полей перед сохранением.
✅ Навигация: Переходы между экранами через Intent.
🛠 Используемые технологии
Язык программирования: Java
Среда разработки: Android Studio
Интерфейс: XML Layouts (Linear, Relative, Constraint), RecyclerView
Хранение данных: SQLite (встроенная база данных на устройстве)
Минимальная версия Android (SDK): 24 (Android 7.0)
💾 Способ хранения данных
В приложении используется локальная база данных SQLite через класс SQLiteOpenHelper.
Это соответствует заданию, так как данные сохраняются в файле базы данных внутри файловой системы телефона и доступны после перезапуска приложения без подключения к интернету.

🚀 Инструкция по запуску
Скачайте исходный код проекта или клонируйте репозиторий.
Откройте Android Studio.
Нажмите File -> Open и выберите папку с проектом.
Дождитесь завершения синхронизации Gradle (Build Sync).
Запустите приложение на эмуляторе (нажмите ▶️ Run) или подключите реальное устройство по USB.
Рекомендуется использовать эмулятор Pixel 6 API 34.

📂 Структура проекта
MainActivity.java — Главный экран со списком заказов и статистикой.
OrderFormActivity.java — Экран создания нового заказа и редактирования.
TrackActivity.java — Экран подробной информации о заказе (детали + кнопки действий).
OrderAdapter.java — Адаптер для отображения списка в RecyclerView.
DatabaseHelper.java — Класс для работы с SQLite (создание таблиц, CRUD-операции).
Order.java — Модель данных (POJO класс описания заказа).
res/layout/ — XML-файлы разметки интерфейса.
res/values/ — Ресурсы (цвета, строки, стили).
